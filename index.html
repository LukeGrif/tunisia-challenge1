Mapping of positions (matches your original Python code):

  • v1, v2, v3 = vertices
  • e12 = edge between v1–v2
  • e23 = edge between v2–v3
  • e31 = edge between v3–v1

Each side sum:
v1 + e12 + v2, v2 + e23 + v3, v3 + e31 + v1

Logic: we iterate all permutations of length 6 from 1..10 and collect those that satisfy all three equalities.
